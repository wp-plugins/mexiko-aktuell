== Description ==

Konfigurierbares Widget, das aktuelle und informative Nachrichten verschiedener Bereiche aus Mexiko bereitstellt.
Die Nachrichten wird als News-Ticker in der Sidebar zur Verf&uuml;gung gestellt.

Bereitgestellt von [http://www.mexiko.mx/](http://www.mexiko.mx/).

Das Mexiko Aktuell-Plugin ist direkt im Widget konfigurierbar.

Das Wordpress-Template muss widget-f&auml;hig sein. Anleitung dazu: [http://automattic.com/code/widgets/themes/](http://automattic.com/code/widgets/themes/)

== Installation ==

1. Entpacke `mexiko-aktuell.1.0.zip`.
2. Lade den `exiko-aktuell`-Ordner in dein Verzeichnis `/wp-content/plugins` hoch.
3. Aktiviere das Plugin im 'Plugins'-Men&uuml; der Wordpress-Administration.
4. F&uuml;ge das Widget unter Design->Widgets deiner Sidebar hinzu und bearbeite die Widget-Optionen.
